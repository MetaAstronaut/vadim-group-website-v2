# SITE STRUCTURE — The Vadim Group

## Purpose
Defines the structure, hierarchy, and content logic of the website.

The Vadim Group website is a premium, service-focused platform presenting residential repair and marine/RV repair services with clear navigation, strong value communication, SEO-oriented structure, and WhatsApp-based conversion flow.

---

Each page must support:

1. Trust building
2. Expertise demonstration
3. Conversion to lead
4. AI-indexable content blocks (SEO + AEO)

---

## 3. WhatsApp Business Integration

### Implementation Points:

**1. Global Header CTA:**
- Desktop: "WhatsApp Us" button next to "Get a Quote"
- Mobile: Floating WhatsApp button (bottom-right corner)

**2. Contact Form Alternative:**
- Every contact form should have: "Prefer to message directly? Start a WhatsApp chat"

**3. Footer:**
- WhatsApp Business link in contact section

**4. WhatsApp Pre-filled Message Template:**

```
Hi! I'm interested in [Service Type] and would like to discuss:
- Issue: [Brief description]
- Location: [City/Area]
- Timeframe: [Urgent/This week/Flexible]
- Photos: [Yes/Will send separately]
```

**5. Response Workflow:**
- Automated initial reply: "Thanks for reaching out! We'll review your message and respond within 2-4 hours during business hours."
- Human follow-up with questions and estimate scheduling
