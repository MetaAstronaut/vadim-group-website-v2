# SITE STRUCTURE — The Vadim Group

## Purpose
This document defines the full information architecture, navigation logic, 
page goals, and CTA flow for The Vadim Group website.

Each page must support:
1. Trust building
2. Expertise demonstration
3. Conversion to lead
4. AI-indexable content blocks (SEO + AEO)


## 1. Core Navigation

Top-level Navbar:
- Home
- Home Repairs
- Specialized Services (dropdown)
- Marine & RV
- Commercial (optional future)
- Blog (later: Knowledge Center)
- Contact
- [CTA] Get a Quote

Footer Links:
- About
- Services Overview
- Marine & RV Services
- Blog
- Contact
- Privacy Policy
- Terms


## 2. Page-by-Page Architecture

### 2.1 Home Page (highest-conversion page)
**Role:** Storytelling + trust building + segmentation + lead capture  
**Primary CTA:** Request a Quote  

**Section Order:**
1. Hero (Trust + Urgency)
2. Value Proposition — "Why We Exist"
3. Services Overview (3 blocks)
4. Differentiation — "Why Clients Choose Us"
5. Micro-Case Examples (photo or text)
6. Process Overview (1–7 steps)
7. Testimonials
8. CTA Section (Quote request)
9. FAQ (short)
10. Mini-footer CTA

**Future Enhancements:**
- Add video / before-after carousel
- Add Marine & RV teaser banner


### 2.2 Home Repairs (Main Service Page)
**Role:** Detailed breakdown + credibility signals  
**Content:** Categorized repair types via accordion  
**CTA:** Request Assessment or Request Estimate  


### 2.3 Specialized Services (Dropdown / Category Page)
**Includes (initial):**
- Structural & foundation
- Electrical repairs (non-hazardous)
- Kitchen & bathroom
- Exterior & interior
- Cosmetic & finishing
- Custom requests

**Role:** On-page filtering and service depth


### 2.4 Marine & RV Services
**Role:** Niche USP + specialization proof + pricing logic + story  
**CTA:** Request Consultation  
**Sections:**
1. Hero + short narrative
2. Why marine repairs require a specialist
3. Types of projects
4. Tools, materials, standards
5. Mini-FAQ for marine & RV
6. CTA

**Once content available → convert to full conversion page + gallery**


### 2.5 Contact Page
**Role:** Form + trust boosters + availability clarity

Required Fields:
- Name
- Phone
- Address / Area
- Description
- Timeline (Urgent / This week / Flexible)

Add elements:
✔ trust badges  
✔ small "How it works" 1-2-3 visual flow  


### 2.6 Portfolio (planned)
**Status:** Hidden until content ready  
**Format:** Case-based storytelling, not gallery only:
- Problem
- Diagnosis
- Solution
- Materials & technique
- Result
- Client reaction


### 2.7 Blog / Knowledge Center
**Phase:** After core site + design  
**Goal:** AEO + SEO + authority building
